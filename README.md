# OverFusion

TAS tool/framework for games based on the MMF2/CTF2.5 runtime

## List of the supported games

- I Wanna be The Boshy
- I Wanna be The Guy
- I Wanna be The Try (1.9.8.3)
- I Wanna be The Try: A New Adventure (DEMO)
- Five Nights at Freddy's
